# docs

Use this directory for durable project knowledge.

## Project: twitter-poster

An agent-assisted project for automating Twitter/X posts.

## Recommended file types

- research notes
- architecture decisions
- harness contract changes
- setup instructions
- API usage notes
- postmortems

## Suggested pattern

- one topic per file
- include date when the document is time-sensitive
- record why a decision was made, not only what changed
